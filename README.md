# 🎬 Media to Video Generator

A powerful web-based application that allows users to upload multiple images and videos and merge them into a single, seamless video output. Built with *ReactJS (Vite)* on the frontend and powered by *FFmpeg* for media processing.

---

## Features

- Upload multiple images and videos  
- Combine media into a single video  
- Fast processing using FFmpeg  
- Supports mixed media formats (images + videos)  
- Flexible media ordering  
- Modern UI built with React + Vite  

---

## Tech Stack

- *Frontend:* ReactJS + Vite  
- *Media Processing:* FFmpeg  
- *Language:* JavaScript / Node.js  

---

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/RaoulBock/MEDIA-TO-VIDEO-GENERATOR-BACKEND.git
cd MEDIA-TO-VIDEO-GENERATOR-BACKEND
npm run dev
```

# Usage

- Open the application in your browser
- Upload multiple images and/or video files
- Click create video
- Will generate your video and automatically download

# Supported Formats

- *Images:* JPG, PNG, JPEG
- *Videos:* MP4

# FFmpeg Requirement

Make sure FFmpeg is installed and accessible in your system path.
